CLARIFY - A Google Chrome Extension

HTMl, CSS, and JavaScript

During HackCamp 2023, Canada's largest beginner-friendly hackathon, me and a couple other friends (Dean, Eric, and Jin) created a Chrome Extension that serves as a translator that uses ChatGPT to translate English idioms into simple sentences.

Learning a new language can be challenging, and effective communication is crucial. The inspiration behind CLARIFY comes from our desire to simplify language barriers, making it more comfortable for individuals to communicate with each other. I worked on the front-end, handling how the responses were displayed using HTML, Tailwind CSS, and the DaisyUI component library. This was my first experience using the OpenAI API so it was cool to learn from my very talented teammates!

You'll find attached a video that we made for the competition, it was loads of fun to make and will give you a better idea of how the extension works! In addition, here are links to the GitHub repository and the DevPost page for our official hackathon submission.
